Copy isi test/tricky-html.txt ke input HTML, lalu coba selector berikut:

*                     -> universal selector, cocok banyak node
div                   -> tag selector, harus cocok #app dan dua .card
.panel                -> class selector, cocok 2 section
#hero                 -> id selector, cocok 1 section
button.primary        -> tag + class, cocok tombol Run
.btn.primary.large    -> multi-class, cocok tombol Run
button[type=submit]   -> attribute + tag, cocok tombol Save
[data-card=promo]     -> attribute tanpa tag, cocok card promo
nav .external         -> descendant combinator, cocok link External
main > section        -> child combinator, cocok dua section langsung di main
ul > li               -> child combinator, cocok 4 item list
li + li               -> adjacent sibling, cocok item ke-2 sampai ke-4
li ~ li               -> general sibling, cocok item ke-2 sampai ke-4
.card .badge          -> descendant combinator, cocok dua badge
.item .highlight      -> descendant + class, cocok span Beta dan Delta
